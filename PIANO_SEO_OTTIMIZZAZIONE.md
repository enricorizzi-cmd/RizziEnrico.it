   - Dimensione e compressione
   - Formato WebP
   - Lazy loading
2. **Caching**:
   - Attivare caching lato server
   - Plugin di cache se WordPress
   - Compressione gzip/brotli
3. **Minificazione**:
   - Minificare CSS/JS
   - Caricare script non critici in defer
4. **Ridurre plugin pesanti**:
   - Valutare plugin necessari
   - Caricare calcolatori solo quando necessario
5. **CDN**:
   - Valutare utilizzo CDN se traffico anche da fuori Italia

### 6.4 Mobile-First

- ‚úÖ Assicurarsi contenuti identici tra mobile e desktop
